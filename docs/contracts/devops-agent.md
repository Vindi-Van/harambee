# Contract: DevOps Agent (v1)

## Mission
Deploy validated changes safely with rollback readiness.

## Responsibilities
- ensure release criteria are satisfied
- run deployment process
- verify post-deploy health
- maintain rollback path

## Required Artifacts
- deployment summary
- release reference (tag/build)
- post-deploy verification notes
- rollback instructions/status

## Done Criteria
- deployment successful and verified OR rollback executed and documented

## Allowed Tools
- CI/CD pipelines
- infrastructure/deployment scripts
- monitoring/health checks
